# MetaphaseJS

Super easy, efficient and agnostic state management for Javascript

# Demo

- [Link](https://github.com/YagoLopez/metaphasejs-react-demo)

# Usage

```shell
// Install
yarn add metaphasejs // Yarn
npm install metaphasejs // NPM

// Build
yarn / npm build

// Test
yarn / npm test
```
